$$

Total derivative:
$$
\frac{df}{dt}
=============

\frac{\partial f}{\partial x}\frac{dx}{dt}

\frac{\partial f}{\partial y}\frac{dy}{dt}

2\pi^2 t(3t^4 + 2t^2 - 1)

## 1️⃣ Expected Value of a Random Variable

Formula:
$$
E(X) = \sum x_i p_i
$$

Condition:

* If $$\sum p_i \ne 1$$ → return `"Invalid input"`
* Output rounded to **2 decimal places**

### ✅ Python Function

```python
def expected_value(data):
    outcomes, probabilities = data

    if len(outcomes) != len(probabilities):
        return "Invalid input"

    if round(sum(probabilities), 5) != 1:
        return "Invalid input"

    exp_val = sum(x * p for x, p in zip(outcomes, probabilities))
    return round(exp_val, 2)
```

### Example

```python
expected_value([[1, 2, 3], [0.2, 0.5, 0.3]])
```

**Output**

```
2.1
```

## 2️⃣ Z-score Calculation

Formula:
$$
Z = \frac{x - \mu}{\sigma}
$$

Condition:

* If $$\sigma \le 0$$ → `"Invalid input"`
* Round to **2 decimal places**

### ✅ Python Function

```python
def z_score(data):
    x, mu, sigma = data

    if sigma <= 0:
        return "Invalid input"

    z = (x - mu) / sigma
    return round(z, 2)
```

### Example

```python
z_score((75, 70, 5))
```

**Output**

```
1.0
```

## 3️⃣ Vector Magnitude and Dot Product

Formulas:
$$
||u|| = \sqrt{\sum u_i^2}
$$

$$
||v|| = \sqrt{\sum v_i^2}
$$

$$
u \cdot v = \sum u_i v_i
$$

Condition:

* If vector lengths differ → `"Invalid input"`

### ✅ Python Function

```python
import math

def vector_operations(data):
    u, v = data

    if len(u) != len(v):
        return "Invalid input"

    mag_u = math.sqrt(sum(x**2 for x in u))
    mag_v = math.sqrt(sum(y**2 for y in v))
    dot_product = sum(x*y for x, y in zip(u, v))

    return (round(mag_u, 2), round(mag_v, 2), round(dot_product, 2))
```

### Example

```python
vector_operations([[1, 2, 3], [4, 5, 6]])
```

**Output**

```
(3.74, 8.77, 32.0)
```
