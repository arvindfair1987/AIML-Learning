# Q1. Conceptual Eigenvector Question

### Original (needs correction)

> Let T be a matrix transformation such that one of its eigenvectors is the vector V and its corresponding eigenvalue is E. Which of the following statements is necessarily true?
>
> It is not necessary that V changes either length or direction on transformation by T. Consider the trivial example of the identity transformation. A vector Equation transformed by it remains unchanged.

### Reframed (clear & correct)

Let (T) be a linear transformation. If (v) is an eigenvector of (T) with eigenvalue (\lambda), which statement is necessarily true?

### Correct Answer

$$[
\boxed{T(v) = \lambda v}
]$$

### Explanation

* An eigenvector is **scaled** by the transformation.
* Direction may change (if (\lambda<0))
* Length may change (if (|\lambda|\neq 1))
* Both may remain unchanged (identity matrix, (\lambda=1))

The given statement about identity transformation is **true but not necessary** for all eigenvectors.

```python
def find_eigenvalues(matrix):
    a = matrix[0][0]
    b = matrix[0][1]
    c = matrix[1][0]
    d = matrix[1][1]

    trace = a + d
    determinant = a*d - b*c
    discriminant = trace**2 - 4*determinant

    sqrt_disc = discriminant ** 0.5
    return [round((trace + sqrt_disc)/2, 2),
            round((trace - sqrt_disc)/2, 2)]
```
